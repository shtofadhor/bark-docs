Inside the World

Five types of structured security intelligence

Policy
Organizational requirement
"The organization shall enforce certificate-based authentication for all service-to-service communication within the production Kubernetes cluster"
Control
Technical safeguard in place
"Network segmentation is implemented between production and development environments using VLAN tagging (802.1Q) with explicit deny-all inter-zone firewall rules"
Finding
Observed gap or weakness
"Container images in the CI/CD pipeline are not scanned for known vulnerabilities before deployment to production registries"
Recommendation
Suggested remediation
"Implement automated container image scanning with severity-based gate policies — block deployment of images with CRITICAL or HIGH CVEs"
Test Scenario
Verification procedure
"Verify that egress traffic from production pods is restricted to explicitly allowlisted destinations via NetworkPolicy enforcement"
